JsonXmlFormat

2013年11月17日


  version 1.2.1

  • Description:

JsonXmlFormat is a tool which can format json and xml.It have the following functions:
    1. Do the accuracy check of json’s or xml’s format and can be formatted output and can customize the output style.
    2. Support UTF-8 coded text, furthermore can not be garbled.
    3. It has the status bar icon, open more simple.
    4. Allow Dock icon to show or hide.
    5. Window can be detached.
    6. Support the color mark.

  • Languages:

English

  • Compatibility:

Mac OS X 10.7 or later
